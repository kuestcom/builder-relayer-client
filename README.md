# Kuest Builder Relayer Client

TypeScript wallet-only client for the Kuest relayer.

## Install

```bash
npm install @kuestcom/builder-relayer-client
```

## Usage

```ts
import { RelayClient } from "@kuestcom/builder-relayer-client";
import { BuilderConfig } from "@kuestcom/builder-signing-sdk";

const builderConfig = new BuilderConfig({
  localBuilderCreds: {
    key: process.env.KUEST_BUILDER_API_KEY!,
    secret: process.env.KUEST_BUILDER_SECRET!,
    passphrase: process.env.KUEST_BUILDER_PASSPHRASE!,
  },
});

const client = new RelayClient(process.env.RELAYER_URL!, 80002, wallet, builderConfig);
const walletAddress = await client.deriveDepositWallet();

await client.deployDepositWallet();
await client.executeDepositWalletBatch(
  [{ target: token, value: "0", data: calldata }],
  walletAddress,
  Math.floor(Date.now() / 1000 + 240).toString(),
);
```

## Public Methods

- `deriveDepositWallet`
- `deployDepositWallet`
- `executeDepositWalletBatch`
- `getNonce`
- `getDeployed`
- `getTransaction`
- `getTransactions`
- `pollUntilState`

Wire auth headers are generated by `@kuestcom/builder-signing-sdk` and remain `KUEST_BUILDER_*`.
